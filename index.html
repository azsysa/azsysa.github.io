Road Assist: Find Nearby Workshops.

Features: Java, Android Studio, Firebase, Google Maps API

Mobile Development.

This project is developed for my final year project. Comes up with two platform which is mobile application is for user and website is for admin.

Road Assist is a mobile application for finding a nearby workshop in Perlis, Malaysia. The apps is designed to display and navigate service provider such as tow truck service, car service and motorcycle service. The workshop data is display and retrieved from database. Available features: login/register, display/filter location, navigate and send sms/call to the service provider.

Web Development.

Web platform is build for admin. All workshop data is controlled by admin and they have features such as view user, delete user, add, edit and delete services.
